# LOTO PREDICTOR PRO

統計傾向をもとに、LOTO 6 / LOTO 7 の予測数字パターンを生成するフロントエンドアプリです。  
あわせて、高額当選実績のある売り場を地図で閲覧できるマップ画面も提供します。

## 特徴

- LOTO 6 / LOTO 7 の条件別パターンをワンクリック生成
- 出現頻度の高い数字を重み付きで抽選
- 合計値・偶奇比・連番条件を満たす組み合わせを優先
- 当選売り場リストとマップ連携（Leaflet + OpenStreetMap）
- モバイル表示に対応したレスポンシブUI

## 画面

- トップ画面: 予測番号を5パターン生成
- マップ画面: 売り場情報の一覧と地図表示

## 使い方

1. このリポジトリをクローン
2. [index.html](index.html) をブラウザで開く
3. `LOTO 6` または `LOTO 7` ボタンを押して予測を表示
4. `高額当選売り場MAP` からマップ画面へ移動

## ファイル構成

- [index.html](index.html): メイン画面
- [script.js](script.js): 予測ロジックとチケット描画
- [map.html](map.html): マップ画面
- [map.js](map.js): 売り場データ処理、座標検索、マーカー描画
- [style.css](style.css): 全体スタイル
- [順位,売り場名,所在地,合計回数.md](順位,売り場名,所在地,合計回数.md): 売り場データの元ファイル

## 技術スタック

- HTML5
- CSS3
- JavaScript (Vanilla)
- Leaflet
- OpenStreetMap / Nominatim API

## 注意事項

- 本アプリの予測結果は娯楽目的です。当選を保証するものではありません。
- 地図座標は Nominatim API の検索結果に依存します。
- 無料APIのため、連続アクセス時は表示まで時間がかかる場合があります。
